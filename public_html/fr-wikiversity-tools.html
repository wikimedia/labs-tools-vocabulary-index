French Wikiversity tools
I developped two simple Python scripts based on Pywikibot:
#fac.py is scanning the faculty namespace (106)
#dpt.py is scanning the department namespace (108)
Each script is collecting a set of data and save it in a Lua module table, respectively Module:Table faculté and Module:Table département.

Those programms don't have any relations whith vocabulary-index so that i will create a proper account for them.

Outils pour le projet fr:Wikiversity
J'ai développé deux scripts Python basés sur Pywikibot:
# fac.py parcours l'espace de nom Faculté (106)
# dpt.py parcours l'espace de nom Département (108)
Chaque script collecte un jeu de données et les savegarde dans une table d'un module Lua ; respectivement Module:Table département et Module:Table faculté.

Ces deux programmes n'ont pas de relation avec vocabulary-index, je créerai un compte particulier pour ces nouveaux outils.
