Indexation du vocabulaire pour la faculté de langues étrangères de la Wikiversité francophone

Vocabulary-index est un nouveau projet sur: https://tools.wmflabs.org/ depuis Janvier 2016.
La solution Git est en place, vous pouvez parcourir le dossier .git via http.
Le projet est disponible sur Gerrit pour révision de code.

Jusqu'à maintenant, le programme fonctionne uniquement en ligne de commande. C'est un programme écrit en Python qui utilise Pywikibot.
Essayez: ./vcb.py -h

Le script principal et les modules se trouvent dans le dossier vcb.
Le programme scanne les pages données en argument, il cherche deux modèles (traduction et prononciation) collecte les couples de mots et établit des listes de vocabulaire pointant vers les items correspondant des projets Wiktionnaires. Les modèles pris en considération pour la recherche du vocabulaire: Traduction(s) et Prononciation(s).
La liste des pages, est obtenue en cherchant les modèles suivants dans la page donnée en argument: Leçon et Département.

Si vous avez besoin de tester le programme, faites le de préférence sur le Département:Portugais où il prétend être implémenté.
./vcb.py Portugais/Grammaire/Nom

Le projet au sein du laboratoire de la Wikiversité ; avec des listes extraites de différents départements de langues étrangères.
